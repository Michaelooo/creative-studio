# Creative Studio 🎬

小说与短剧创作项目

## 目录结构

```
creative-studio/
├── novel/              # 小说创作
│   ├── drafts/         # 草稿
│   ├── chapters/       # 章节
│   └── world-building/ # 世界观设定
├── short-drama/        # 短剧创作
│   ├── episodes/       # 分集剧本
│   └── storyboards/    # 分镜脚本
├── characters/         # 角色设定
│   ├── main/           # 主角
│   └── supporting/     # 配角
├── scripts/            # 剧本
│   ├── drafts/         # 草稿
│   └── final/          # 定稿
└── assets/             # 素材资源
    ├── images/         # 图片
    ├── audio/          # 音频
    └── video/          # 视频
```

## AI 辅助工具

本项目使用 AI 辅助创作：
- **分镜脚本生成**：使用 OpenClaw 的 storyboard skill
- **文生图提示词**：遵循结构化提示词框架
- **图生视频提示词**：确保跨镜头一致性

## 提示词框架

### 文生图格式
```
{景别}，{角色外观}，{主体描述}，{环境场景}，{情绪氛围}，{视觉风格}，{光影效果}
```

### 图生视频格式
```
{镜头描述}，{镜头运动}，{主体行为}，{背景变化}，{环境变化}
```

## 规则

1. 每个镜头严格控制在 10 秒以内
2. 相邻镜头组必须在不同空间开场
3. 跨镜头保持角色外观一致性
4. 所有提示词使用中文
